Doctype

<!DOCTYPE> is the first thing that must appear in an html document. It is essential for the browser to know what version of HTML the page is written in.

<!DOCTYPE html> tells the browser it's HTML 5

Html

contains all html elements except for the Doctype tag

Head

<head> is where all the head elements of the webpage go, this includes elements like style, metadata and page title.


<head></head>

Body

this is where all the visible webpage elements go, this is the main part of your website. Images, text, tables, your content goes in here.

<body> This is where your site content goes </body>

Title

Gives your page a title, this shows up at the top of browser window/tab, bookmarks and more. This is an essential part of your site identity.

Tables & Caption

tables encompass a number of tags but these all contribute to creating tables in html.

  • <table> announces the start of a table and you can style the table with this tag.

  • <tr>creates a table row
  • <td>creates a table cell
  • <th>defines a header cell
  • <caption>lets you give the table a caption

<table>
<caption>This is a table caption</caption>
<tr>
<th>This is a header cell</th>
<th>This is also a header cell</th> </tr>
<tr> <td>This is a normal cell</td> <td>this is also a normal cell</td> </tr>
</table>

Meta

Metadata compiles information about your site that a visitor might not necessairly see, this information is useful for browsers, search engines and more. The kind of information stored as metadata includes things like key words, the author of the site and when it was last updated. Metadata is always placed within the <head>.

<meta name="description" content="introduction to HTML">
<meta name="keywords" content="HTML,tags,educational">
<meta name="author" content="Richard Sheehan-Corkery">
